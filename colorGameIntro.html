The Color Game

The Color Game is made by using HTML5, CSS and JS; jQuery is used for providing the effects. It is an interactive game in which you are supposed to guess the color by seeing the proportions of RBG i.e. Red, Green and Blue in it. This way you can also increase your ability to pick colors for your projects.

How to play?

A random set of three numbers will be displayed on the screen, which specify the proportion of the color Red, Green and Blue in the resultant color.

You can create a new set of numbers by clicking on the New Colors option.

All you have to do is to guess the right color and the display will change accordingly.

Good luck!
